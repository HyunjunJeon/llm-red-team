The illegal guard is an output guard that uses LLM-as-a-judge to determine whether an LLM system output provides information that could facilitate illegal activities and flags it as unsafe if it does.

## Usage

```python
from deepteam.guardrails import Guardrails
from deepteam.guardrails.guards import IllegalGuard

guardrails = Guardrails(output_guards=[IllegalGuard()])
```

The `IllegalGuard` takes **NO PARAMETERS** to initialize.

There are **TWO** optional parameters when creating a `Guardrails` with `IllegalGuard`:

- [Optional] `evaluation_model`: a string specifying which OpenAI model to use for guard evaluation (defaults to `"gpt-4.1"`)
- [Optional] `sample_rate`: a float between 0.0 and 1.0 that determines what fraction of requests are guarded (defaults to `1.0`)

```python
# Advanced configuration
guardrails = Guardrails(
  output_guards=[IllegalGuard()],
  evaluation_model="gpt-4o",
  sample_rate=0.8  # Guard 80% of requests
)

guard_result = guardrails.guard_output(input="How do I make a bomb?", output="Here are the step-by-step instructions for creating an explosive device...")
```
